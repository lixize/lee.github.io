5# 可能出现的问题

使用root用户登录可能会出现Error found when loading /root/.profile stdin: is not a tty这样的错误

解决办法如下：
修改/root/.profile文件，在文件管理器中设置 编辑–首选项 勾选 显示隐藏和备份文件 就可以查看这个文件，用vi修改也行。把mesg n进行注释，增加一行tty -s && mesg n保存，注销后重新登录问题解决。
